The attached code was created to help with data processing of direct infusion-style (no chromatography) hydrogen-deuterium exchange mass spectrometry data.
Commercial software that target analysis of HDX-data has a different structure and processing method, compared to what is required for experiments without chromatograms
Originally, it aimed to work with LAP-LDI MS created data, which has 2 dimensions- scan numbers, and m/z spectra.
